# CENTRAL ARCHIVE

Bu klasör, AgentFrame içindeki tüm önemli operasyonel geçmişi **merkezi** olarak arşivlemek için kullanılır.

## Kapsam
- soru / cevap geçmişi
- önemli prompt kayıtları
- hata / çözüm geçmişi
- git diff / commit / sürüm geçmişi
- karar geçmişi

## Amaç
Parçalı log yapısını tek merkezde toplayıp hem insan hem LLM için geçmişi izlenebilir kılmak.
