# PROJECT BOOTSTRAP RULES

Yeni proje açılırken uygulanacak kurallar:

## 1) Yeni Repo Aç
- proje kendi GitHub repo’sunda açılır

## 2) Seed Edilecekler
- `workflow/`
- gerekli `scripts/`
- giriş dosyaları (`AGENTFRAME_QUICKSTART.md` benzeri proje quickstart)

## 3) AgentFrame’de Kalacaklar
- framework geliştirme dosyaları
- ortak reusable scriptler
- framework dashboard / kurallar / stratejiler

## 4) Proje Repo’da Olacaklar
- ürün kodu
- proje task/planning
- proje deploy/logs/metrics

## 5) Bootstrap Çıktısı
Yeni repo şu minimum yapıyla başlar:
- `workflow/`
- `src/`
- `tests/`
- `deploy/`
- `logs/`
- `metrics/`
