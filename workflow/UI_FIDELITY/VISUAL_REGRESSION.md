# VISUAL REGRESSION POLICY

## Baseline Yönetimi
- Baseline yalnızca onaylı UI değişikliklerinde güncellenir.
- Baseline değişimi tek başına PR açıklamasında gerekçelendirilir.

## Karşılaştırma
- Desktop + mobile kırılımları ayrı kıyaslanır.
- Dinamik alanlar (tarih/saat) maskelenir.

## Rapor
- Diff görselleri PR artefact olarak saklanır.
